1. Download the zip file and extract the content.
2. Place the Play_Tennis_Data_Set.csv and RuleMining.py under same folder.
3. Point the folder containing python file and run the python file.
4. On running it, the program will ask for the minimun support value.
5. After providing the minimum support value and Minimun Confidence value, the program generates a text file "Rules.txt" containing the sales above the minimun support and confidence value provided.
