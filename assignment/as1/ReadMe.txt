1. Download the zip file and extract the content.
2. Place the Car_Sales_Data_Set.csv and assignment2_code.py under same folder.
3. Point the folder containing python file and run the python file.
4. On running it, 3 csv files each containing sorted data will by generated.
5. A list of options will be displayed to select the sales details based on various attributes. Select the needed option to know the sales details.
